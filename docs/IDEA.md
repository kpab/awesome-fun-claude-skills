# アイデア: awesome-fun-claude-skills (仮)

## コンセプト
ネタ系・エンタメ系の Claude Code スキル(ペルソナ・ジョーク・遊び系)を集約する awesome リスト + 閲覧サイト。

## 背景・動機
- 汎用のスキル集約(awesome-claude-skills 67k★ 等)は激戦区だが、すべて実務・生産性特化でネタ系カテゴリは扱っていない
- ネタ系特化の集約はほぼ無人(claude-code-personas 0★、claude-personalities 4★ 程度)
- 一方で素材は散在して豊富。特に日本語圏はキャラ系スキル文化が強い(ギャル・ござる・関西のおばちゃん・執事・推しキャラ等が個別記事としてバズるのに、集約する場所がない)
- 既存の巨大リストを「量」で超えるのではなく、「ネタ系特化」というニッチで一番になる戦略

## ラフスコープ
- やる: ネタ系スキルのキュレーション(GitHub リポジトリを収集・分類。記事のみのスキルは対象外)
- やる: awesome-* 形式の GitHub リポジトリ(英語ベース、日本語スキルも掲載)
- やる: 閲覧サイト(Astro + Cloudflare 想定)
- やらない(今回は): 自作スキルの大量生産(キュレーションが主軸。自作は初期コンテンツの一部として)
- やらない(今回は): 投稿・レビュー機能などの動的機能

## 競合・差別化(調査済み: 2026-07-10)
- 直接競合: なし(ネタ系特化の集約は事実上存在しない)。判定: **空いている**
- 間接競合: 汎用 awesome リスト群(ComposioHQ 67k★、VoltAgent 28k★、travisvn 14k★)とスキル検索サイト(claudeskills.info、claudemarketplaces.com、agent-skills.cc)— いずれも実務特化でネタ系カテゴリなし
- 差別化: ネタ系特化。出典の言語圏を問わず、質(スター数・再現性・面白さ)で選ぶグローバルリスト(掲載は GitHub リポジトリのみ)

## 検討した他の候補
- claude-side-quests: キャッチーだが検索流入が弱いため見送り(サイト名としての再利用余地あり)
- awesome-claude-personas: ペルソナ特化に絞りすぎてジョーク・遊び系をこぼすため見送り
- plugin marketplace 形式: 配布主体になるため見送り(将来 リポジトリに marketplace.json を足す拡張は可能)
